# licenser

simple utility to write `LICENSE` files to your current directory. Currently supports the following licenses:
```
AGPL-3.0
GPL-3.0
GPL-2.0
LGPL-3.0
LGPL-2.1
NON-AI-MPL-2.0
MPL-2.0
NON-AI-APACHE-2.0
Apache-2.0
NON-AI-UNLICENSE
UNLICENSE
NON-AI-MIT
MIT
```
